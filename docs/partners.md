# Partners

The Partners view provides tools for browsing, searching, and managing media partner information.

## Overview

This section allows you to:

- Browse all media partners
- Search and filter by various criteria
- View detailed partner profiles
- Manage partner preferences and settings

## Partner List

### Browsing Partners

The main view displays partners in a searchable list:

- Partner name
- Office affiliation
- Quality tier
- Publication rate
- Last assignment date

### Sorting Options

Sort the list by:

| Column | Description |
|--------|-------------|
| Name | Alphabetical order |
| Tier | Quality tier (A+ to C) |
| Office | Geographic office |
| Pub Rate | Publication success rate |
| Last Active | Most recent assignment |

### Filtering

Filter partners by:

- **Office**: Los Angeles, New York, Chicago, etc.
- **Tier**: A+, A, B, C
- **Affiliation**: Network, freelance, etc.
- **Status**: Active, inactive, under review

## Partner Profile

Click any partner to view their full profile.

### Basic Information

- **Name**: Partner/outlet name
- **Contact**: Email, phone
- **Address**: Physical location
- **Office**: Assigned geographic office
- **Affiliation**: Network or freelance

### Quality Metrics

#### Tier

Partners are ranked into tiers:

| Tier | Description |
|------|-------------|
| A+ | Premium partners with exceptional publication rates |
| A | High-performing, consistent partners |
| B | Standard partners with reliable output |
| C | Developing or lower-volume partners |

#### Publication Rate

Rolling 24-month publication success:

- Percentage of loans resulting in clips
- Based on at least 3 observed loans
- Weighted toward recent performance

### Approved Makes

List of vehicle makes the partner can receive:

- Make name
- Quality rank (1-5, higher is better)
- Preference indicator

Example:
```
BMW        ★★★★★
Mercedes   ★★★★☆
Porsche    ★★★★☆
Toyota     ★★★☆☆
```

### Preferred Days

Days the partner prefers for:

- **Pickup**: When they want to receive vehicles
- **Dropoff**: When they return vehicles

Example:
```
Preferred Pickup: Monday, Wednesday
Preferred Dropoff: Friday
```

### Assignment History

Recent assignments with:

- Vehicle make/model
- Dates
- Publication result
- Notes

## Managing Partners

### Editing Partner Info

1. Open partner profile
2. Click **Edit**
3. Modify fields
4. Save changes

!!! note
    Some fields sync from FMS and cannot be edited locally.

### Updating Preferred Days

1. Open partner profile
2. Go to Preferences section
3. Select new preferred days
4. Save

### Changing Tier

Tier changes typically require approval:

1. Request tier change through FMS
2. Provide justification
3. Wait for approval
4. Change reflects in scheduler

### Adding Approved Makes

1. Open partner profile
2. Go to Approved Makes
3. Click **Add Make**
4. Select make and rank
5. Save

## Partner Status

### Active vs Inactive

| Status | Description |
|--------|-------------|
| Active | Available for assignments |
| Inactive | Not currently receiving vehicles |
| Under Review | Temporarily suspended pending review |

### Review Status

Partners may be placed under review for:

- Low publication rates
- Quality issues
- Contract negotiations
- Capacity concerns

### Reactivating Partners

1. Review the issue
2. Resolve concerns
3. Update status in FMS
4. Partner becomes available in scheduler

## Search Tips

### Quick Search

Type in the search box to find by:

- Partner name (partial match)
- Contact email
- City

### Advanced Search

Use filters for complex queries:

- Tier A+ partners in Los Angeles
- Partners with BMW approval
- Inactive partners with recent history

## Bulk Operations

### Select Multiple Partners

Use checkboxes to select multiple partners.

### Available Bulk Actions

- **Update Preferred Days**: Set for all selected
- **Export List**: Download as CSV
- **Assign to Office**: Reassign geographic office

## Analytics

### Partner Performance

View aggregated metrics:

- Average publication rate by tier
- Geographic distribution
- Make preferences distribution

### Trends

Track partner metrics over time:

- Publication rate changes
- Assignment frequency
- Tier movements

## Best Practices

### Regular Review

- Check inactive partners monthly
- Review low publication rate partners
- Update preferences as needed

### Data Quality

- Keep contact info current
- Verify addresses for distance calculations
- Update approved makes list

### Tier Management

- Monitor publication rates for tier adjustments
- Document tier change justifications
- Track tier movements over time
